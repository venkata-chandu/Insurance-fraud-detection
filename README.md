# Insurance Fraud Detection System

This project uses Machine Learning (Random Forest) to detect fraudulent insurance claims.

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- Tailwind CSS

## Model Performance
- Accuracy: 79%
- Fraud Recall: 69%
- Threshold: 0.30

## How to Run
1. Create virtual environment
2. Install requirements
3. Run app.py
## Datasets used
- Dataset link:-https://www.kaggle.com/datasets/buntyshah/auto-insurance-claims-data
